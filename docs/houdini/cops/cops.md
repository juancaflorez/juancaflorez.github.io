## Use ANY uv coordinate attribute inside COPs

To convert uv into flat 2d space, throw a uv node and sample uv0 in COPs using an attrib import (labs)

[Getting geometry attributes in COP networks in Houdini](https://www.youtube.com/watch?v=9qYUho0lWns)

Used to convert from equirectangular to planar projection without baking
