Pieces | Code Project

Date

Jun 16, 2020

Categories

Coding, shaders

Is a code project inspired by the watercolors of Thomas Girtin and other prominent painters from the 18th century. In this project I avoid using textures and instead each element in the game is made out of small flat pieces with solid colors.

First, each object is sampled using a Poisson distribution pattern, this is done in python using the Open3d library. The next step is to use those samples inside Blender to generate the parts and create a nested hierarchy of objects. To use all the parts inside a game engine the hierarchy is encoded in a low-resolution texture using a fork of the pivot painter tool for blender. My new fork allows multiple branching hierarchies and control points to control how complete an object looks.
