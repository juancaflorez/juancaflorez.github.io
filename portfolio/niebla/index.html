Niebla | Game Prototype

Date

Jun 16, 2020

Categories

Code, prototype, unreal, mechanic

Fog of War

Niebla is a game prototype about using the fog of war mechanic from traditional RTS games and adding dynamic elements to it. The game reacts by remembering how much time the player has spent in a given area, it tries to delay the advance of the player and accounts for secret paths inside the fog of war.

The technical implementation of the game consists of using a render target and graph to represent the state of the map, the graph itself is a Delaunay/Voronoi graph storing the neighbors and locations of various samples, each sample accounts for important areas of the level. The render target is driven by the graph but is updated using a shader to manage the smooth transitions between the states of the samples.

Art style

The art style uses negative space and flat areas as a design element to suggest incompleteness and hidden paths in the game.

Grease pencil

Controller

Engine

The game is made using Unreal Engine, mostly done using blueprints for quick iteration. For the more demanding task like building the graph and the various point generation algorithms C++ was used.
